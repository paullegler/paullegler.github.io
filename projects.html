Projects

1. Basketball Shot Chart
Takes in text documents with information about a players shots (x, y, result) where x and y describe the location of the shot on the court and result says whether the shot made or missed. Given a large database of shot spreedsheats, this could create shot charts for each player. If the database of shots contains Excel files with the important information, I wrote a python script to turn these Excel spreadsheets into correctly formatted text documents.
